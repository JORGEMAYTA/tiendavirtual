## Instalacion

antes de empezar recordar tener ya instalado
- Xampp
- Composer 
- Nodejs
- PHP 8.2 Minimo

Para entornos de desarrollo, ejecutar dentro de la carpeta despues de haber clonado el repositorio...

```sh
composer install
npm install
```
